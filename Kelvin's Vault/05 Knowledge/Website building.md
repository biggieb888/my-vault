1. CONTEXT    — What is this site/page for? Who is the audience?
2. STRUCTURE  — What sections/components do you need?
3. STYLE      — Aesthetic direction, colors, fonts, vibe
4. BEHAVIOR   — Any interactivity? Hover states, animations, mobile behavior?
5. CONSTRAINTS — Tech stack (plain HTML? React?), any existing code to match?

AI can't read your mind. The more precisely you describe aesthetics, the better the output. Learn these concepts:

- **Typography:** serif vs sans-serif, font weight, line height, letter spacing
- **Spacing:** padding (inside), margin (outside), gap (between grid/flex items)
- **Color:** hex codes, opacity/alpha, gradients (linear vs radial)
- **Visual hierarchy:** what draws the eye first, second, third
- **Responsive design:** how a site adapts from desktop → tablet → mobile (breakpoints)
